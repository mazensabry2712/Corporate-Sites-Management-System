How It Works:
1. In Select Dropdowns (e.g., PR Number, Project Name):
<select name="filter[pr_number]">
    <option></option>
    @foreach($prNumbers as $prNumber)
        <option value="{{ $prNumber }}" {{ request('filter.pr_number') == $prNumber ? 'selected' : '' }}>
            {{ $prNumber }}
        </option>
    @endforeach
</select>
2. In Text/Number Inputs (e.g., Value Min, Deadline):
<input type="number"
       name="filter[value_min]"
       value="{{ request('filter.value_min') }}">
3. Active Filters Counter:
@if(request()->has('filter'))
    <span class="active-filters-badge">
        {{ count(array_filter(request('filter', []))) }} Active
    </span>
@endif
4. Active Filters Summary (NEW):
@foreach(request('filter', []) as $filterKey => $filterValue)
    @if(!empty($filterValue))
        <span class="badge badge-primary">
            {{ ucfirst(str_replace('_', ' ', $filterKey)) }}:
            <strong>{{ $filterValue }}</strong>
        </span>
    @endif
@endforeach
